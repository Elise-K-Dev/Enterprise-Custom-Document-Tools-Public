# Open WebUI Tool Imports

These JSON files are sanitized examples for registering the public tool servers.

Included:

- `openwebui-rust-tools.json`
- `openwebui-python-tools.json`
- `openwebui-markdown-pdf-tools.json`
- `openwebui-web-tools.json`

Not included:

- Private review tools.
- Music-generation integrations.
- Account bootstrap scripts.
- Runtime tokens or user grants.
